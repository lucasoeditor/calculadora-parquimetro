# 🅿️ Calculadora de Parquímetro

Uma aplicação simples em JavaScript que simula o funcionamento de um parquímetro, calculando o tempo de estacionamento com base no valor inserido pelo usuário e retornando também o troco, quando necessário.

👉 https://lucasoeditor.github.io/calculadora-parquimetro/

[![Abrir no navegador](https://img.shields.io/badge/Acessar-Projeto-green)](https://lucasoeditor.github.io/calculadora-parquimetro/)

---

## 🚀 Funcionalidades

- Inserção de valor pelo usuário  
- Cálculo automático do tempo de estacionamento  
- Exibição do tempo adquirido  
- Cálculo e exibição do troco  
- Validação de valores inválidos

## 💡 Lógica básica

O sistema segue uma tabela de valores típica de parquímetro:

- R$1,00 → 30 minutos  
- R$1,75 → 60 minutos  
- R$3,00 → 120 minutos  

Caso o valor inserido não seja suficiente, o sistema informa o erro.
